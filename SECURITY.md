# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please email `githubkaissuer@gmail.com` with details. Do **not** open a public issue for security concerns.

## Supported Versions

We aim to respond to reports within 72 hours and provide a fix or mitigation when feasible.

## What to Include

Please include:
* Description of the vulnerability
* Steps to reproduce (if applicable)
* Impact and potential risks
